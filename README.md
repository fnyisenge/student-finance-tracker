# Student Finance Tracker

This is a web application for tracking personal student finances. You can add, edit, delete, and search transactions. It includes a dashboard showing total records, total amount, and top category. Data is persisted using localStorage.

## Features

- Add, edit, and delete transactions
- Live regex search
- Dashboard metrics
- Local storage persistence
- Accessible design (skip links and ARIA updates)

## Technologies

- HTML, CSS, JavaScript (ES Modules)
- LocalStorage for persistence

## How to Use

1. Open `index.html` in a browser.
2. Add a transaction in the "Add Record" form.
3. View records in the "Records" section.
4. Search using regex in the search input.
5. Dashboard updates automatically.
