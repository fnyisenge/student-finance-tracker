About Student Finance Tracker

The Student Finance Tracker is a web application designed to help students manage daily expenses, track spending patterns, and monitor financial habits.

Features

  • Add, edit, and delete transaction records
  • View records in an organized table
  • Automatic storage in your browser using localStorage
  • Dashboard summarizing total spending
  • Accessible and responsive design
